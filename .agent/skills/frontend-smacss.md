# Skill: Frontend SMACSS & JS ES6
- **CSS Architecture:** Dividir archivos por componentes (Base, Layout, Component, State, Theme).
- **Naming BEM:** Usar `bloque__elemento--variante`. Evitar selectores de ID o anidación de más de 3 niveles.
- **JS Moderno:** - Obligatorio File-closure: `(() => { ... })();`.
    - No usar `var`. Usar `const` y `let`.
    - Comparaciones estrictas (`===`).
    - jQuery: Prefijar variables con `$` y usar `.prop()` para booleanos.
