# Skill: Quality & Integrity Standards
- **Accesibilidad (WCAG):** - Todo HTML debe tener roles ARIA y labels.
    - Mantener contraste 4.5:1.
    - Seguir guías de "Accessibility Standards for Drupal".
- **Composer:** - El nombre del paquete en `composer.json` DEBE ser `drupal/[machine_name]`.
    - Los componentes internos se nombran `drupal/parent-component`.
- **Limpieza:** Eliminar etiquetas `@author` y `@version` de todos los archivos.
